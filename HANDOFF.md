## Project summary

Testing whether candidates who receive more media attention tend to win elections.

Two pre-registered hypotheses:
- **H1**: The candidate with more mentions wins >50% of the time (binomial test vs. 0.5)
- **H2**: Mention share positively correlates with vote share (Pearson r)

All figures are generated by `scripts/generate_figures.py`. All data lives in `data/raw/`. Only real data is used — no synthetic data. Run the script with `.venv/bin/python3 scripts/generate_figures.py` from the project root.

## Data collected so far

## Governor data collection (DONE — was the blocking task)

The old `collect_ngrams_governors.py` used Harvard Dataverse file IDs that returned Senate data. MEDSL has **no** candidate-level governor dataset (verified by searching the `medsl` Dataverse — it has President/Senate/House constituency returns only). Klarner's "Governors Dataset" (doi:10.7910/DVN/PQ0Y1N) has governor *names* but no opponent/vote-share pairs.

**Solution used:** `scripts/collect_governors_wikipedia.py` scrapes Wikipedia race-page infoboxes (`{year} {state} gubernatorial election`). For each year it lists the category members, fetches infobox wikitext, parses nominees/parties/percentages (handles `{{nowrap}}`, `{{percent|v|t}}` templates, state party names, nickname/suffix stripping), and keeps races where the top two are one D and one R. Output: `data/raw/governors_cleaned.csv` (801 D-vs-R races, 1960–2018, 50 states; territories dropped).

`collect_ngrams_governors.py` now reads that CSV, maps state names → postal codes, and runs the same `ngrams_pair()` loop. `ngram_term()` strips suffixes (Jr./III) and converts `'W. P. "Bill" Atkinson'` → `'Bill Atkinson'` for queryable terms. Result: **753 usable, H1=69% (p<0.0001), H2 r=0.240 (p<0.0001)**.

Governors are now integrated as a 4th level in Figs 4, 6, and 7. (Figs I1/I2 are international-only — no governor panel.) While there, fixed a latent Fig 4 Panel A bug where jittered scatter numeric x-positions polluted the category axis; Panel A is now a numeric axis with explicit ticks.

## Key code notes

- `generate_figures.py` imports all CSVs at the top. Senate vote_pct is stored as a **fraction** (0.54), presidential as a **percentage** (49.7). Code converts pres to fraction with `/ 100`.
- Color convention: D=`#1060C8` (blue), R=`#C82010` (red). Attention winner=`#2c7873` (teal), upset=`#f4a261` (amber). Do NOT use red/blue for attention outcomes.
- OLS r values are annotated inline near the regression line using `ols_label()` helper — not in the legend.
- Fig 7 is a single-panel H1 forest plot (panel A only). B/C/D were intentionally removed.
- The `.venv` Python environment has all dependencies. Always use `.venv/bin/python3`.
